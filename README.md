# 🚢 Cruise Process Wiki

> End-to-end **L1 → L2 → L3 → L4** process documentation for cruise & integrated resort operations.
> Reference company: **Royal Caribbean International** (Royal Caribbean Group, NYSE: RCL)
> Live wiki: **https://ghatk047.github.io/Cruise-Process-Wiki/**

## Overview

The Cruise Process Wiki documents **228 business processes** across **12 operational domains** covering every aspect of cruise ship and shore-side operations — from guest embarkation and stateroom housekeeping to marine engineering, environmental compliance, and digital guest experience.

Each subprocess page includes:
- **BPMN Diagram** — Pre-rendered PNG (3840×2160, generated by `mmdc` from Qwen-authored Mermaid source)
- **Swim Lane Chart** — Visual role/responsibility matrix
- **L4 Process Steps** — 10-12 detailed steps with Role, System, Input, Output, KPI, Decision Point, Exception, Pain Point
- **Process Attributes** — Identification, trigger, outcome, KPIs, RCCL-specific risks

python3 scripts/generate_cruise_wiki.py
